# 示例与拆解

## 五、完整示例

### 示例一：电影场景图

```text
【画面目标】 生成一张适合作为电影关键帧的古风悬疑场景图。
【核心题材】 古风、雨夜、宫城、刺客、叙事瞬间。
【画面一句话】 暴雨夜，一名黑衣刺客站在古城屋脊上，俯视远处灯火通明的皇宫。
【镜头角度】 低机位侧后方仰视，摄像机贴近湿滑屋瓦，镜头略微向上倾斜。
【镜头尺寸】 中长焦电影镜头，轻微空间压缩，让刺客、屋脊和远处宫殿处于同一紧张空间中。
【画幅比例】 2.39:1宽银幕。
【景别】 复合构图，刺客占画面右侧下方约20%，远处皇宫占画面中央偏左区域。
【构图】 前中后景分层，屋脊形成引导线，将视线带向远处皇宫。
【景深】 中等浅景深，刺客轮廓与屋瓦雨水清晰，远处皇宫灯火略微虚化。
【主体一——黑衣刺客】
【核心身份】 夜行刺客。
【体型/外貌/轮廓】 背对镜头，披着湿透的黑色斗篷，轮廓锐利，身体微微前倾。
【材质/纹理】 黑色旧布、湿润皮革、金属短刃，表面有雨水反光。
【动作/状态】 静止凝视远处皇宫，右手握住短刃。
【画面作用】 叙事主角，制造悬念。
【主体二——皇宫】
【核心身份】 戒备森严的古代宫城。
【外貌/材质/动作】 屋檐层叠，宫墙深红，灯火在雨幕中模糊发亮。
【画面作用】 目标地点，也是危险来源。
【核心机位位置】 古城屋脊近处，贴近湿瓦。
【环境/场景】 古代都城，夜晚，雨水覆盖屋顶和石墙。
【天气与氛围】 暴雨夜，空气潮湿，紧张、隐秘、危险。
【前景】 湿滑屋瓦和雨滴轻微虚化。
【中景】 黑衣刺客站在屋脊上。
【远景】 皇宫灯火、城墙和雨雾。
【背景】 深蓝黑夜空与雨幕，无清晰星空。
【主光源/方向】 皇宫灯火来自画面左前方，微弱冷月光来自右上方。
【光线软硬质感】 低照度侧逆光，雨水反射形成细碎高光。
【辅助光/互动光】 刺客肩部和短刃边缘有微弱轮廓光。
【特效光/气氛光】 雨幕中的灯火轻微光晕溢出，克制不夸张。
【整体色温/基调】 冷调为主，局部暖灯点缀。
【主色系统】 深蓝黑、湿瓦灰、宫墙暗红、暖黄色灯火。
【色彩饱和体系】 低饱和，局部灯火中等饱和。
【高光与阴影倾向】 高光集中于雨水、刀刃、远处灯火；阴影集中于刺客服装和屋檐下方。
【对比度与质感】 高明度对比，低色彩对比，电影剧照质感。
【细节密度】 中高细节，主体轮廓清楚，背景细节被雨雾柔化。
【最终画面质感】 写实电影剧照，克制、真实、紧张。
```

### 示例二：产品商业图

```text
【画面目标】 生成一张高级科技耳机产品主视觉图。
【核心题材】 科技产品、商业摄影、极简背景。
【画面一句话】 一副银白色无线耳机悬浮在干净的灰白空间中，透明充电盒微微打开。
【镜头角度】 平视三分之四视角，略高于产品中心线。
【镜头尺寸】 中长焦商业摄影镜头，减少透视变形，保持产品比例准确。
【画幅比例】 1:1方图。
【景别】 近景，产品占画面中央约60%。
【构图】 中心构图与留白构图，产品居中，四周保留干净空间。
【景深】 浅景深，耳机主体与充电盒边缘清晰，远处背景柔和虚化。
【主体一——无线耳机】
【核心身份】 高端主动降噪无线耳机。
【体型/外貌/轮廓】 两只耳机轻微悬浮，曲线圆润，轮廓完整。
【材质/纹理】 哑光银白金属、细腻磨砂塑料、黑色声学网孔。
【动作/状态】 静态展示，耳机从充电盒中轻微升起。
【画面作用】 商品主体，展示设计与材质。
【主体二——透明充电盒】
【核心身份】 半透明科技感充电盒。
【外貌/材质/动作】 磨砂透明外壳，边缘有柔和高光，盒盖微微打开。
【画面作用】 补充产品功能与科技感。
【核心机位位置】 产品正前方的棚拍机位。
【环境/场景】 极简灰白商业摄影空间，无杂乱元素。
【天气与氛围】 干净、安静、精密、高级。
【前景】 无明显前景，仅有柔和地面反射。
【中景】 耳机与充电盒。
【远景】 灰白渐变背景。
【背景】 纯净灰白背景，无文字、无复杂道具。
【主光源/方向】 大面积柔光来自画面左上方。
【光线软硬质感】 商业棚拍柔光，阴影柔和且可控。
【辅助光/互动光】 右侧弱轮廓光勾勒耳机边缘，底部有轻微反射补光。
【特效光/气氛光】 不强调特效光，只保留真实产品反光。
【整体色温/基调】 中性偏冷，干净现代。
【主色系统】 银白、浅灰、黑色细节、透明磨砂质感。
【色彩饱和体系】 低饱和，高洁净度。
【高光与阴影倾向】 高光集中在耳机边缘、充电盒透明转折处；阴影非常柔和。
【对比度与质感】 中等明度对比，高级商业摄影质感。
【细节密度】 中等细节，产品关键结构清楚，背景极简。
【最终画面质感】 真实高级产品摄影，干净、精密、可用于商品主图。
```

## 六、原始沙虫案例为什么有效

原始案例强的地方不只是“写了沙虫”，而是它把画面拆成了清楚的视觉系统：

- 低机位仰视：让沙虫巨口更有压迫感。
- 超长焦与宽银幕：把战士、沙尘、巨口压缩在同一空间里，形成电影大场面。
- 多主体尺度对比：战士不是主角细节，而是尺度参照，衬托沙虫巨大。
- 明确焦点：焦点锁定沙虫巨口内部黑暗区域，观众第一眼知道看哪里。
- 前中后景：前景战士、中景空洞、远景沙雾，空间层次稳定。
- 材质具体：没有只写“怪物皮肤”，而是写成风化岩层、矿物、干裂地壳，让画面更真实。
- 光线具体：侧逆光、沙尘散射、微弱边缘光，避免普通平光。
- 色彩克制：低饱和、沙尘米色、深土褐、纯黑，统一了画面情绪。

你以后写提示词时，可以记住一个简单公式：

```text
画面目标 + 镜头位置 + 主体关系 + 空间层次 + 光线方向 + 材质纹理 + 色彩基调 + 最终质感
```
