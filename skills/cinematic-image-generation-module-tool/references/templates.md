# 模板参考

## 一、万能主模板

下面是可复制的完整框架。方括号 `[填写：...]` 是需要替换的地方；如果某一项与你的画面无关，可以删除。

```text
【画面目标】 [填写：这张图最终要表达什么，例如电影海报感、产品介绍图、角色设定图、建筑空间概念图、故事关键帧]

【核心题材】 [填写：古风/科幻/现实/奇幻/商业产品/建筑/人物/怪物/游戏概念/其他]

【画面一句话】 [填写：用一句话描述画面，例如“雨夜古城屋脊上，一名黑衣刺客俯视灯火通明的皇宫”]

【镜头角度】 [填写：低机位仰视/平视/高机位俯视/鸟瞰/近距离贴地/过肩视角/主观视角/侧面观察等]。[填写：镜头是否向上、向下、倾斜，角度约多少]

【镜头尺寸】 [填写：广角/标准镜头/中长焦/超长焦/微距/电影宽银幕等]。[填写：空间是开阔、自然、压缩、夸张还是贴近]

【画幅比例】 [填写：2.39:1宽银幕/16:9/4:3/1:1/3:4竖图/9:16竖屏]

【景别】 [填写：远景/全景/中景/近景/特写/大特写/复合构图]。[填写：主体在画面中占比，例如底部15%-20%、中央三分之一、几乎填满画面]

【构图】 [填写：中心构图/三分法/对称构图/框架式构图/引导线构图/前中后景分层/留白构图/压迫式构图]。[填写：画面视觉中心在哪里，观众第一眼看哪里]

【景深】 [填写：浅景深/中等景深/深景深/焦点堆叠/背景虚化/前景轻微失焦]。[填写：焦点锁定在谁或哪里]

【主体一——[填写：主体名称]】
【核心身份】 [填写：人物/生物/建筑/产品/道具/自然物/机械/城市/场景核心的身份定位]
【体型/外貌/轮廓】 [填写：大小、形状、比例、轮廓、姿态、视觉识别特征]
【材质/纹理】 [填写：皮肤、金属、布料、玻璃、岩石、木材、陶瓷、液体、尘土、旧化痕迹等]
【动作/状态】 [填写：站立、奔跑、悬浮、坠落、爆发、凝视、静置、展示、打开、破损、发光等]
【画面作用】 [填写：主角/尺度参照/视觉焦点/威胁来源/情绪核心/商品主体/背景符号]

【主体二——[填写：可选主体名称]】
【核心身份】 [填写：第二主体身份，没有可删除]
【外貌/材质/动作】 [填写：它与主体一的关系，例如对峙、陪衬、互动、遮挡、指向、形成尺度对比]
【画面作用】 [填写：增加故事、制造对比、解释空间、衬托主角、形成商业卖点]

【主体三——[填写：可选主体名称]】
【核心身份】 [填写：第三主体身份，没有可删除]
【外貌/材质/动作】 [填写：必要信息]
【画面作用】 [填写：必要信息]

【核心机位位置】 [填写：镜头位于哪里，例如地面低处、街角、室内门口、产品正前方、山崖边、人物肩后、空中俯拍]

【环境/场景】 [填写：地点、时代、空间类型、文明痕迹、自然环境、室内外关系]

【天气与氛围】 [填写：晴天、阴天、雨夜、雾、沙尘、雪、烟、霓虹湿气、晨雾、战场烟尘等]。[填写：情绪，例如庄严、孤独、紧张、神秘、安静、奢华、危险]

【前景】 [填写：前景有什么，是否虚化，是否遮挡，是否作为尺度参照]

【中景】 [填写：主要事件或主要物体所在区域]

【远景】 [填写：背景空间、天际线、山脉、城市、建筑、群众、天空、虚化光点等]

【背景】 [填写：背景是否清晰，是否有天空/地平线/墙面/纯色背景/环境细节]

【主光源/方向】 [填写：自然日光/窗光/霓虹灯/火光/棚拍柔光/顶光/侧光/逆光/侧逆光]，光线来自[填写：画面方向]。

【光线软硬质感】 [填写：硬光/柔光/柔化硬光/漫反射/强方向光/低照度/高反差/均匀商业光]

【辅助光/互动光】 [填写：边缘光、反射光、屏幕光、火光、霓虹反光、产品轮廓光、人物面部补光]

【特效光/气氛光】 [填写：体积光、空气中漂浮颗粒、薄雾发亮、雨滴反光、尘埃光束、光晕溢出；如果不需要可写“克制，不强调明显特效光”]

【整体色温/基调】 [填写：冷调/暖调/冷暖对比/中性色/低色温/高色温/复古胶片/现代商业干净色]

【主色系统】 主色为：[填写：2-4个主色，例如沙尘米色、深土褐、纯黑、金属银、雾蓝、暗红]

【色彩饱和体系】 [填写：极低饱和/低饱和/中等饱和/高饱和/局部高饱和点缀/褪色胶片感/商业高洁净色]

【高光与阴影倾向】 高光集中于：[填写：受光边缘、眼睛、产品反光、雨水、金属边缘等]。阴影集中于：[填写：背景深处、主体背光面、建筑内部、画面角落等]。

【对比度与质感】 [填写：高明度对比/低色彩对比/高反差黑白/柔和低对比/厚重真实/干净精致/纪录片式/高级商业摄影/游戏概念设定]

【细节密度】 [填写：极简/中等细节/高细节/局部精细、整体克制/复杂但有主次]

【最终画面质感】 [填写：真实摄影、电影剧照、概念艺术、商业摄影、产品渲染、纪实摄影、胶片颗粒、写实插画、游戏原画]
```

## 二、画面任务分支模板

分支模板不是按题材硬分类，而是按“这张图要完成什么画面任务”分类。古风、科幻、游戏、商业都可以套进同一个任务类型里。

### 1. 宏大场景型

适合：电影大场面、战争、灾难、神话、科幻城市、古风战场、末日景象、巨物压迫。

重点字段：

- 【镜头角度】：低机位仰视、高机位俯视、远距离长焦、鸟瞰。
- 【景别】：远景、全景、复合构图。
- 【主体关系】：至少保留主体一和主体二，用大小差制造尺度感。
- 【前中后景】：必须填写，空间层次越清楚越稳定。
- 【光线】：侧逆光、体积光、雾气、尘埃、雨雪能增强大场景纵深。

追加句式：

```text
【尺度关系】 [填写：小主体] 被压缩在 [填写：巨大主体/环境] 之下，仅作为尺度参照存在，强调空间体量与人类渺小感。
【空间纵深】 画面由前景[填写]、中景[填写]、远景[填写]形成清晰层次，远处逐渐被[填写：雾/烟/雨/沙尘/光晕]吞没。
```

### 2. 人物主体型

适合：人物肖像、角色设定、古风人物、科幻人物、职业形象、情绪肖像、游戏角色。

重点字段：

- 【镜头角度】：平视最自然，低机位更英雄，高机位更脆弱。
- 【景别】：半身、中近景、近景、特写。
- 【主体材质】：服装、发型、皮肤、配饰、武器、职业符号。
- 【眼神/姿态】：人物图必须写清楚。
- 【光线】：面部主光、轮廓光、眼神光。

追加句式：

```text
【眼神与表情】 [填写：冷静/疲惫/警觉/温柔/愤怒/空洞/专注]，视线朝向[填写：镜头/画外/下方/远处]。
【服装与身份符号】 [填写：服装层次、材质、配饰、武器、徽章、职业道具] 清晰可见，但不抢夺面部焦点。
```

### 3. 空间建筑型

适合：建筑外观、室内空间、古城、未来基地、商业空间、遗迹、街区、展厅。

重点字段：

- 【镜头角度】：广角、平视、低机位、对称构图、鸟瞰。
- 【构图】：对称、引导线、框架式构图很重要。
- 【材质】：石材、混凝土、木材、玻璃、金属、灯光反射。
- 【人/物尺度】：加入小人物或家具帮助判断空间大小。

追加句式：

```text
【空间结构】 [填写：拱门/走廊/中庭/台阶/穹顶/立面/街巷/展台] 形成清晰的透视引导线，将视线导向[填写：视觉中心]。
【尺度参照】 画面中加入[填写：人物/车辆/家具/树木/灯具]作为尺度参照，使建筑体量更明确。
```

### 4. 生物怪物型

适合：怪物、神兽、异形、巨兽、动物、幻想生物、生态生物设定。

重点字段：

- 【核心身份】：要写清楚它像什么生态位，而不只是“怪物”。
- 【轮廓】：远看先成立，细节再补充。
- 【材质】：皮肤、鳞片、甲壳、羽毛、骨骼、矿物、湿润或干燥。
- 【尺度关系】：最好加入环境或人物。

追加句式：

```text
【生态逻辑】 该生物像真实生态系统中的[填写：捕食者/深海生物/沙漠生物/森林伏击者/远古巨兽]，外形细节服务于它的生存方式。
【轮廓识别】 即使在远景或雾中，也能通过[填写：角、背脊、翅膀、巨口、肢体比例、尾部]识别它。
```

### 5. 产品商业型

适合：产品图、商品海报、品牌视觉、科技产品、食品饮品、奢侈品、包装图。

重点字段：

- 【景别】：近景、特写、中近景。
- 【构图】：中心构图、对称构图、留白构图。
- 【材质】：产品表面必须写清楚。
- 【光线】：商业柔光、轮廓光、反射控制。
- 【背景】：干净，不要让环境抢主体。

追加句式：

```text
【产品呈现】 [填写：产品名称] 位于画面视觉中心，轮廓完整清晰，品牌识别区干净，表面材质与关键卖点被光线准确强调。
【商业光线】 使用[填写：大面积柔光/边缘轮廓光/顶部柔光/反射板补光]，让产品边缘、材质、透明度、金属反射或纹理清晰可见。
```

### 6. 概念设定型

适合：游戏概念图、世界观设定、载具设定、道具设定、服装设定、科幻/古风/奇幻设计稿。

重点字段：

- 【核心身份】：说明它在世界观里的功能。
- 【设计语言】：时代、文化、科技水平、材质体系。
- 【细节密度】：局部高细节，整体轮廓清楚。
- 【展示角度】：正面、侧面、三分之四视角、剖面、陈列式。

追加句式：

```text
【设计逻辑】 所有结构细节服务于[填写：战斗/仪式/交通/居住/采矿/医疗/宗教/权力象征]功能，避免无意义装饰。
【设定可读性】 保持整体轮廓清晰，关键功能区域可辨认，细节集中在[填写：核心结构/接口/武器/纹样/材质转折处]。
```

### 7. 叙事瞬间型

适合：电影剧照、故事插画、关键事件、人物互动、动作瞬间、悬疑场景。

重点字段：

- 【画面一句话】：必须像一句故事。
- 【主体关系】：谁在做什么、谁看向哪里、冲突是什么。
- 【时间点】：动作发生前、发生中、发生后。
- 【氛围】：紧张、安静、危险、悲伤、荒诞等。

追加句式：

```text
【叙事时刻】 画面捕捉在[填写：事件发生前一秒/动作爆发瞬间/冲突结束之后/真相即将揭露时]，所有人物姿态与光线都服务于这一瞬间。
【视线关系】 [填写：主体一] 看向[填写：主体二/画外/某个物体]，形成明确的故事指向。
```
